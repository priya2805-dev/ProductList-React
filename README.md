# Product Page React

**Responsive React Product Page with Grid, Forms, and Tables**

---

## Overview
A fully responsive product page built with **React.js** and **CSS**.  
Features include:

- Product grid layout with hover effects  
- Responsive form for adding/editing products  
- Styled table with scroll on small screens  
- Mobile-first design for desktop, tablet, and mobile  

---

## Technologies
- React.js  
- CSS3  
- JavaScript (ES6+)  
- HTML5
